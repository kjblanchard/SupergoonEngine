# Supergoon Engine
- Cross Platform 2d engine,  written in C and C++ (for Imgui visual editor)

## Details
- Builds on Linux, Macos (universal), Windows, iOS, Android, Emscripten


## Installation

## Libraries | Components
- SDL3 (windowing, input, renderer)
- lua (scripting)
- freetype (text)
- ogg (sound)
- vorbis (sound)
- imgui (debug editor)
