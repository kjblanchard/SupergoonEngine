# Supergoon Engine

- Cross Platform 2D game library

## Details

- Builds on Linux, Macos (universal), Windows, iOS, Android, Emscripten
- Wraps a bunch of libraries and tries to make it "easy" to work with them.

## Installation

## Libraries | Components

- [SDL3 (windowing, input)]("https://github.com/libsdl-org/SDL")
- [OpenGL (3.3/ES)]("https://www.opengl.org/")
- [OpenALSoft]("https://openal-soft.org/")
- [freetype (text)]("https://freetype.org/")
- [ogg (sound)]("https://www.xiph.org/ogg/")
- [vorbis (sound)]("https://www.xiph.org/vorbis/")
