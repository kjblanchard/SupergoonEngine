# Supergoon Engine

- Cross Platform 2d engine,  written in C

## Details

- Builds on Linux, Macos (universal), Windows, iOS, Android, Emscripten

## Installation

## Libraries | Components

- SDL3 (windowing, input)
- OpenGL (3.3/ES)
- OpenALSoft
- lua (scripting)
- freetype (text)
- ogg (sound)
- vorbis (sound)
